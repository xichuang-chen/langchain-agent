# QA

### 1. agents.ts 中 `const prompt = await pull<ChatPromptTemplate>("hwchase17/react");` 什么意思？  
- 作用：从 LangChain 的社区或官方模板库拉取一个预设的 React Agent Prompt 模板
- 返回值：ChatPromptTemplate 对象  
它包含了：
    - 系统消息（System Prompt）
    - 工具说明（Tools Description）
    - 用户对话模板

- 目的：不用自己手写复杂的 Prompt，让 LLM 能按照 ReAct（Reason + Act）模式 决策调用工具

- 可以理解成：
“给我一份已经设计好的 Agent 大脑提示模板，我直接拿来用”
